# LastFmHonorific

Plugin do Dalamud (FFXIV) que atualiza o seu título do **Honorific** com a música
que está tocando "agora" no seu perfil do **Last.fm** — funciona com qualquer
player que faça scrobbling, desde que o Last.fm esteja recebendo os scrobbles.

## Proveniência e créditos

A adaptação foi feita inteiramente com ajuda de um assistente de IA (Claude, da Anthropic),
a partir do código-fonte público do SpotifyHonorific.
